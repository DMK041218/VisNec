Abstract

Selecting examples where vision actually matters.

Multimodal instruction datasets contain many samples where the visual input contributes unevenly. VisNec measures visual necessity by comparing model loss with and without visual information, then prioritizes samples where image evidence is most useful for instruction tuning.
